# Meme Maker
A simple meme generator made for fun, using canvas and Node.js for the backend.

## Usage
Clone or download the repo.
Then start the server-side:
```sh
$ cd meme-maker
$ npm install
$ npm start
```
Navigate to the page, upload your image and write some funny lines.
